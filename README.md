# 💱 Currency Converter App

A responsive **Currency Converter Web Application** built using **HTML, CSS, and JavaScript** that allows users to convert currencies easily through a clean and interactive interface.

## 🚀 Features

- Convert between multiple currencies
- Real-time currency conversion
- Responsive and user-friendly UI
- Fast and interactive experience
- Built using pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
